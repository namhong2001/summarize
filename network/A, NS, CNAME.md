A, NS, CNAME
===========
* A - DNS 에서 요청한 hostname의 ip주소를 알려준 경우
* NS - DNS 에서 DNS query를 날릴 다른 DNS를 알려준 경우
* CNAME - DNS 에서 NS와 같이 다른 DNS 주소를 알려주면서, 
요청한 hostname의 공식 이름(canonical name)을 알려준 경우
